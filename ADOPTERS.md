## How to add yourself

1. Open a pull request that adds a row above. The columns:
   - **Organization / project** — link to your site or repo if you'd like
   - **Backend(s)** — one or more of `crontab`, `systemd-timer`,
     `kubernetes`, `aws-scheduler`, `vercel`
   - **Use case** — one sentence on what cronix runs for you
   - **Contact** — GitHub handle, email, or "private" if you'd rather not
     publish one

2. If you can't make your usage public, email **abellaismail@gmail.com**
   instead. We can list you as "Confidential adopter (Fortune 500
   logistics company)" or similar, with no other identifying detail.

## Why this matters

A real adopter list is a hard requirement for the [CNCF Sandbox](./docs/cncf-sandbox-application.md)
application and a strong signal that the project is solving real
problems. Names in this file get their open issues weighted ahead of
speculative roadmap work.

# Code of Conduct

cronix adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
in full. The canonical text and FAQ live at that URL.

In short: we expect everyone in this project — maintainers, contributors,
users, and visitors — to treat one another with respect, both in this
repository and in any public space where they represent the project.

## Reporting

If you experience or witness behavior that violates the Code of Conduct,
contact **asabani.work@gmail.com**.

All reports are handled confidentially. Maintainers will acknowledge
receipt within 72 hours and follow up with the reporter on next steps.

Possible outcomes include a private warning, a public warning, a
temporary or permanent ban from project spaces, or no action if the
report is unsubstantiated. The decision rests with the maintainers
collectively; the reported party is not part of the decision.

## Scope

This Code applies in all project spaces: this repository, any issue
tracker or discussion forum operated by the project, project chat
channels, and public events where someone represents cronix.

## Enforcement guidelines

We follow the [Contributor Covenant enforcement ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines):
correction → warning → temporary ban → permanent ban, escalating based
on severity and repetition.
